### 创建线程只有一种方式
创建Thread的实例

#### 创建并运行Java运行时代码（线程任务）的三种方式
- 继承Thread类，覆写其run方法，run方法会在调用start()方法之后被执行
- 实现Runnable接口，实现run方法，run方法会在调用start()方法之后被执行
- 实现Callable接口，实现其call方法，call方法会在调用start()方法之后被执行
